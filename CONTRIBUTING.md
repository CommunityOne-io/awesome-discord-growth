# Contributing

Thanks for helping make this list better!

**Rules**
- Concise description (≤100 chars), neutral tone
- Use **https**, prefer official/maintained sources
- Sort **alphabetically** within sections
- No affiliate links or tracking params
- New categories must start with ≥3 credible links

**How**
1. Fork and branch: `feat/add-<tool-name>`
2. Edit `README.md`, place link in the right section A→Z
3. (Optional) Run `npx awesome-lint`
4. Open a PR with a short rationale
